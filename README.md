# Flying_Robotics
Командная задача НТО по профилю "Летающая робототехника" команды "Полетели"

## Задача
Дрон Клевер должен автономно пролететь вдоль нефтяного трубопровода, обнаружить все врезки (боковые ответвления трубы) и сообщить их координаты.

## Установка и запуск

Клонируйте репозиторий в домашнюю директорию:
```bash
cd ~
git clone https://github.com/Max-lab10/Flying_Robotics.git
```

Перейдите в папку проекта и запустите скрипт инициализации:
```bash
bash start.sh
```

Веб-интерфейс: создайте текстовый документ, вставьте туда код из `scripts/Front-end_drone_v4`, сохраните как `.html`. Откройте файл в браузере.

## Структура проекта

| Файл | Назначение |
|---|---|
| `start.sh` | Инициализация: копирование launch, генерация мира, запуск симулятора и нод |
| `simulator.launch` | ROS launch-файл для симуляции дрона Клевер в Gazebo |
| `scripts/gen_world.py` | Генерация случайного мира с нефтяным трубопроводом и врезками |
| `scripts/pipeline_inspector.py` | Нода автономного полёта: навигация вдоль трубы, обнаружение врезок через камеру, публикация координат |
| `scripts/color_branch_detector.py` | Дополнительная нода: цветовое обнаружение врезок |
| `scripts/Front-end_drone_v4` | Веб-интерфейс Mission Control Panel |

## ROS-топики

| Топик | Тип | Направление | Описание |
|---|---|---|---|
| `/uav/pose` | PoseStamped | Subscribe | Позиция дрона |
| `/main_camera/image_raw` | Image | Subscribe | Изображение с камеры |
| `/detected_branches` | PoseArray | Publish | Обнаруженные врезки (основная нода) |
| `/detected_branches_color` | PoseArray | Publish | Обнаруженные врезки (цветовая нода) |
| `/mission/status` | String | Publish | Статус миссии |
| `/mission/start` | Empty | Subscribe | Команда старт |
| `/mission/land` | Empty | Subscribe | Аварийная посадка |
| `/mission/kill` | Empty | Subscribe | Kill switch |

## Логика миссии

1. **Взлёт** — дрон поднимается на высоту 1.5 м над первым сегментом трубы
2. **Полёт вдоль трубы** — последовательный перелёт к каждому сегменту трубопровода
3. **Обнаружение врезок** — обработка изображения с камеры в реальном времени
4. **Уточнение** — пролёт к известным позициям врезок для проверки
5. **Возвращение и посадка**
